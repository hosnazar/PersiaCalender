# PersiaCalender v6

نسخه PWA محصولی‌تر برای اجرا روی iPhone/Safari و GitHub Pages.

## قابلیت‌ها
- باز شدن روی تاریخ امروز
- تقویم شمسی با نمایش میلادی و مناسبت‌ها
- تعطیلات رسمی با علت تعطیلی و برچسب منبع
- یادآور تکی و تکرارشونده با صدای انتخابی در محدوده قابلیت PWA
- ویجت آب‌وهوا با انتخاب واقعی شهر/روستا از Open-Meteo
- ویجت اذان بعدی و اوقات شرعی از Aladhan API
- روزشمار تولد/سالگرد/مناسبت
- قبله‌نما با GPS
- تنظیمات فونت، زوم، تراکم، تم، شهر پیش‌فرض، پشتیبان‌گیری و بازیابی
- About Us: designed by Hossein Nazari / hosnazar@gmail.com

## محدودیت صریح
برای آلارم پس‌زمینه صددرصد، ویجت واقعی iOS Home Screen، Live Activity و Dynamic Island باید نسخه Native SwiftUI ساخته شود. PWA این سطح دسترسی iOS را ندارد.
